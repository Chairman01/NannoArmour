# Nano Armor — Website

Marketing website for **Nano Armor**, a Calgary, Alberta roof rejuvenation and protection company.
Advanced nano-technology that restores and protects aging asphalt shingles — a cost-effective
alternative to full roof replacement.

## Structure

- `index.html` — Home
- `services.html` — Services
- `about.html` — About
- `contact.html` — Contact + quote form
- `styles.css` — Shared styles
- `robots.txt`, `sitemap.xml` — SEO
- `assets/` — Images, video, favicon, flags

Plain HTML/CSS. No build step required.

## Deploy (Vercel)

1. In Vercel: **Add New → Project → Import** the `NannoArmour` GitHub repo.
2. Framework Preset: **Other**. No build command. Click **Deploy**.
